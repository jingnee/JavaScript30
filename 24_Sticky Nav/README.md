순서

```
1. 네비게이션 바의 상단 위치찾기
2. 스크롤하다가 네비게이션바의 상단을 넘어갈때 네비게이션바를 고정시킴
3. 고정시킬때, 본문과 네비게이션바 사이에 네비게이션바의 높이만큼 간격생성
4. 고정시킬때, 로고 나오게하기
```



### 1. 네비게이션 상단위치 찾기

```javascript
const nav = document.querySelector('#main');
const topOfNav = nav.offsetTop;
```

id가 `main`인 naviation bar를 가져오고, 상단의 위치를 topOfNav에 넣는다.



### 2. 스크롤하다가 네비게이션바의 상단을 넘어갈때 네비게이션바를 고정시킴

```javascript
function fixNav(){
    if(window.scrollY >= topOfNav){
      document.body.classList.add('fixed-nav');
    }else{
      document.body.classList.remove('fixed-nav');
    }
  }

  window.addEventListener('scroll',fixNav);
```

scroll 이벤트가 발생할때마다 `fixNav` 함수를 실행시킨다.

이 함수는 window의 scrollY의 값이 네비게이션 상단바의 위치를 넘어가면! 네비게이션을 상단에 고정시키는 `fixed-nav`라는 클래스를 추가시킨다. 만약 위치를 넘어가지 않는다면 그 `fixed-nav`스타일을 삭제해준다.

그러면 이제 CSS에서 `fixed-nav` style을 만들어보자

```css
.fixed-nav nav{
  position : fixed;
  box-shadow: 0 5px rgba(0,0,0,0.1);
}
```

`fixed-nav`라는 이름의 스타일 클래스는 이미 존재하는(만들어져 있음) `nav`에 적용시킨다.

위치는 고정시키고, 아래에 5px만큼 연한회색으로 그림자를 만들어 준다.

여기까지하고 스크롤을 내려보면 네비게이션바는 고정되지만, 고정되는 순간 기존의 위치를 차지하기 때문에, 본문과의 간격이 갑자기 좁아지는것을 볼 수 있다. 

그래서 네이게이션바의 높이만큼 공간을 추가해줘야 자연스럽게 된다.

### 3. 고정시킬때, 본문과 네비게이션바 사이에 네비게이션바의 높이만큼 간격생성

```javascript
if(window.scrollY >= nav.offsetTop){
    document.body.style.paddingTop = nav.offsetHeight + 'px';
}
else{
    document.body.style.paddingTop = 0;
}
```

fixNav안에 있는 if문에 body의 padding값을 nav의 높이만큼 추가해준다.

아닌경우 원래대로 0으로 만들어준다.

### 4. 고정시킬때, 로고 나오게하기

```css
.fixed-nav li.logo {
  max-width: 500px
}
.fixed-nav .site-wrap{
  transform : scale(0.98);
}
```

네비게이션바를 고정시킬때 로고를 고정시켜주고, 본문의 크기를 조금더 키움
